## Overview

Discussion forums package. Provides threaded message boards with topics,
posts, and forum containers. Extends LibertyContent for post content.

## Key Files

| File | Purpose |
|------|---------|
| `forums/BoardsPost.php` | Post content class |
| `forums/BoardsTopic.php` | Topic container |
| `forums/BoardsForum.php` | Forum container |
| `forums/templates/` | Smarty templates |
